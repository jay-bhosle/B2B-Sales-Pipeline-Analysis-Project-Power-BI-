# B2B Sales Pipeline Analysis (Power BI)

This project delivers an end-to-end sales analytics dashboard built in Power BI for a B2B computer hardware company. The goal is to evaluate sales performance, identify pipeline bottlenecks, analyze revenue trends, and provide actionable business recommendations for leadership.

- Evaluate overall sales performance and revenue trends
- Analyze individual sales agent efficiency and win rates
- Identify underperforming agents and pipeline bottlenecks
- Assess product-level win rates and revenue contribution
- Enable leadership to make proactive, data-driven decisions

---

## üóÇ Dataset Overview

The dataset represents a B2B sales pipeline and includes:
- Sales opportunities and deal stages
- Revenue and deal size
- Sales agents and regions
- Products and opportunity outcomes
- Created vs. closed dates for trend analysis

## üßπ Data Cleaning & Preparation

To ensure reliable KPIs and insights, the following steps were performed:
- Removed duplicate dimension records
- Handled missing values for region, industry, and sales agents
- Standardized region names and sales stage labels
- Removed invalid opportunities (null or zero deal values)
- Fixed date inconsistencies between created and closed dates

These steps ensure that all reported metrics accurately reflect business performance.
